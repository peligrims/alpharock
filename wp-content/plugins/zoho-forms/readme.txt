== Description ==

With Zoho Forms drag-and-drop interface, building forms is easier than ever. Create forms for your website and keep in touch with your visitors. 

Our integration with Zoho's suite of applications like Zoho CRM, Zoho Desk, Zoho SalesIQ, and Zoho Sheet helps you automate data flow between apps and work efficiently.[Click here](https://www.zoho.com/forms/integrations.html?src=WordPress) to learn more about all our integrations.

**Watch the video:**

[youtube https://www.youtube.com/watch?v=8iymV94yoZY]

To embed a form on your WordPress site, you need a [Zoho Forms](https://www.zoho.com/forms/?src=WordPress) account.

[Login](https://www.zoho.com/forms/signup.html?src=WordPress) to your Zoho Forms account and create a form for your WordPress site. Install the Zoho Forms plugin for WordPress and activate the plugin.

Click the Zoho Forms icon under **Embed**. You will be prompted to sign in to your Zoho Forms account (if you haven't already) and choose a form or enter the form permalink URL without signing in. 

Once you define the height and width of your form and click **Embed**, the form's preview will be shown in the editor. 

These setup instructions are for WordPress versions 5.0 and above. 
[Click here](https://www.zoho.com/forms/help/wordpress-plugin.html#WordPress-4.9.9) to learn more about Zoho Forms plugin for WordPress 4.9.9 & below.


== Installation ==

You can install the Zoho Forms plugin two ways:

   1. Install from the WordPress plugin installer.

        * Under the Plugins section of your WordPress Dashboard, search for Zoho Forms.
        * Install via the built-in WordPress plugin installer.

   2. Download the plugin.

        * Access the Plugins tab in WordPress, and search for Zoho Forms.
        * Download the plugin into your system.
        * Upload it to the folder, "/wp-content/plugins/zohoforms/".

Once you install the plugin, you need to activate it to embed your forms. To activate the Zoho Forms plugin, go to the [Plugins](http://codex.wordpress.org/Administration_Panels#Plugins) section in the admin panel and click on [Activate](http://codex.wordpress.org/Plugins_Screen) . Once activated, the Zoho Forms icon will appear in your WordPress editor.

 

== Frequently Asked Questions ==


= How can I change the height and width of my form? =

    * You can edit the height and width while choosing the form you want to embed.

    * You can also modify the existing height and width in the short code that you add in the WordPress editor.

= Do you provide premade templates? =

We have created numerous templates for different verticals. You can customize these templates to suit your needs. [Know more](https://www.zoho.com/forms/templates.html)

= Maximum number of submissions for a form? =

It depends on the plan you have chosen. You can get atleast 500 submissions for a form per month. [Know more](https://www.zoho.com/forms/pricing.html).

= Can I customize my form? =

We provide a number of cutomizing options. You can chose from the list of themes provided or you can create your own. [Know more](https://www.zoho.com/forms/help/tools/themes.html).
